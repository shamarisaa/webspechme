Riwayat Penggunaan

    FAQ Singkat

    Apa yang bisa dilakukan aplikasi ini?

    Aplikasi dapat menerjemahkan suara ke teks, membacakan teks, dan mendeteksi gesture tangan sederhana menggunakan kamera. Cocok untuk membantu komunikasi penyandang tuna rungu.

    Bagaimana cara menggunakan deteksi gerakan jari?

    Buka tab "Gerakan Jari", aktifkan kamera, lalu arahkan satu tangan ke depan kamera. Sistem akan mengenali huruf berdasarkan posisi jari yang terangkat.

    Bisakah data training disimpan?

    Pada versi prototipe ini, deteksi menggunakan heuristic berbasis posisi jari. Untuk akurasi lebih tinggi, bisa dikembangkan dengan TensorFlow.js dan dataset kustom.

    Apakah data saya aman?

    Seluruh pemrosesan dilakukan di browser Anda (lokal). Tidak ada data yang dikirim ke server. Kamera hanya digunakan secara lokal untuk deteksi.

    Browser apa yang didukung?

    Chrome dan Edge memberikan pengalaman terbaik karena mendukung Web Speech API dan MediaPipe. Firefox mendukung sebagian fitur.

    Prototipe untuk demonstrasi. Dapat dikembangkan lebih lanjut dengan model ML terlatih.
